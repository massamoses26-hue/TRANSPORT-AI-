Le projet TRANPORT-AI consiste à estimer le temps T que pourait mettre un tansport sur une distance D à l'aide de l'intelligence artificielle .

Les fonctions de prédictions IA prennent en paramètres des datasets contenues dans des fichiers .csv ayant pour colones respectives :	Heure_Depart, Vitesse_Moyenne,
Distance, Temps_Estimé .

Le repository est composé de trois(3) fichiers :
      test1.ipynb : montre la conception des différentes variables et données utilisées pour le projet ,
                    la conception et l'usage des différentes fonctions utlisées pour la conception des données, la création de modèles IA prédéfinis de python
                    ainsi que les tets de fonctions de ces modèles  avec les résultats adéquats.
      test1.py : contient le code proppre des différentes fonctions utlisées dans le test1.ipynb .
      Transport_Set.csv : est le dataset utlisé sous forme de fichiers .csv pour le projet TRANPORT-AI .


Ce projet a à titre personnel de montrer aux informaticien de l'IT IA l'importance de comprendre l'importance de la compréhension et doncc du choix du type de models Ia en 
fonction de la structure de données .
En occurence ici les modèles de regressions basés sur les Random Forest ( Forêts aléatoires ) et Linear Regression( Regression Linéaire ) nous présentent une différence notable de 
performance sur le même dataset ( à voir dans le test1.ipynb ) .

Pour les curieux(ses) et passionné(e)s de l'IA, nous publierons très prochainement pour une expérience approfondie deux projets :
      L'un sur la regression basé sur les Random Forest ( logique du modèle, code du modèle, explication intuitive des différentes formules et 
      logiques mathématiques derrière ce modèle) 
      L'autre sur la regression basé sur les regressions linéaires ( logique du modèle, code du modèle, explication intuitive des différentes formules et 
      logiques mathématiques derrière ce modèle) .

Alors, n'hesitez à nous suivre !
